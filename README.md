# Project : DietFlow 
This is a service-based website designed to provide users with a customized diet plan based on their health goals and preferences.

Here is the step-by-step workflow:

- Data Collection: Users provide basic information about their health, lifestyle, and dietary goals (e.g., weight loss, muscle gain).

- AI-Generated Plan: The website uses AI to analyze this information and generates a personalized Indian diet chart tailored specifically to the user's needs.

- Instant Delivery: The generated diet plan is delivered directly to the user via WhatsApp or Email in a convenient PDF format.

## Auth System
A clean and responsive Authentication system built with React and Tailwind CSS.

## Features
- Sign In & Sign Up toggle using Conditional Rendering.
